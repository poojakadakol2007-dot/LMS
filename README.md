# 📚 PitStop Library Management System

A modern **Library Management System (LMS)** built using **React + TypeScript + Firebase** with features like book management, admin dashboard, payment integration UI, purchase history, analytics histogram, and responsive ecommerce-style design.

## ✨ Features
- 📖 Add, Update, Delete Books
- 👨‍💼 Admin Dashboard with Analytics
- 💳 Book Purchase & Payment Page
- 📊 Monthly Popular Books Histogram
- 🌙 Dark Mode Support
- 🔥 Firebase Integration
- 📱 Fully Responsive UI
- 🛒 Recently Purchased Books Storage

## 🛠️ Tech Stack
- React + TypeScript
- Vite
- Firebase
- Tailwind CSS
- Chart.js / Recharts

## 🚀 Installation
```bash
git clone https://github.com/your-username/pitstop-library.git
cd pitstop-library
npm install
npm run dev
```

## 📂 Project Structure
```bash
src/
 ├── components/
 ├── lib/
 ├── App.tsx
 ├── main.tsx
 └── index.css
```

## 🔐 Environment Setup
Create a `.env` file and add Firebase credentials:
```env
VITE_FIREBASE_API_KEY=your_api_key
VITE_FIREBASE_AUTH_DOMAIN=your_domain
VITE_FIREBASE_PROJECT_ID=your_project_id
```
